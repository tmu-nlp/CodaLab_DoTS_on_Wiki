Evaluation Criteria

We use D-SARI (based on SARI) as our evaluation criteria for the primary track and F1 score for the supportive track (if the latter exist). We adopt NLTK for both English and German sententce splitting and word tokenization, Bunkai and MeCab for Japanese. (SpaCy for German is too slow so that we abandoned.)

Additionally, we gradually the mean squared error (MSE) away from 150 characters on JADOS.

SWiPE Klexikon JADOS
Primary Track D-SARI D-SARI D-SARI + MSE
Supportive Track F1 None F1

Submission Format

We suggest to use create.submission.py to pack up your prediction into a submission file for all corpora. Python it for its usage or check Participate >> Get Data.

For each corpus, the submission format is a folder containing all generated summary files with an optional deletion prediction file. You can choose to submit any tracks for any corpora.
